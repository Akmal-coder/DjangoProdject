# Интернет-магазин на DjangoProject:
Это проект интернет-магазина товаров (каталог). 

## Описание:
Проект развивается в рамках курса по Django. Включает главную страницу с каталогом и страницу контактов.

## Установка:
1.Клонируйте репозиторий: git clone <ваш-репозиторий>.
2.Запустите сервер: python manage.py runserver.
3.Пройдите по ссылке http://127.0.0.1:8000
